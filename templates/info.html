Welcome to the information page.

This is an educational project simulating an application management process for new potential students.

The site is designed to resemble an actual app more than a typical browsing website. The main focuses were:

  • tidyness or lack of clutter - in order for the user to not be flooded by too many things on the screen
  • mobile first - it is not the best Responsive Design but it's a decent attempt. Depending on what browser you are on, you can Google the steps to see the website in Responsive Design mode. On Firefox you can go to Menu > Web Developer > Responsive Design Mode
  • productivity - the information can be easily copied and pasted in both mobile and desktop mode by the user
